# 📚 Book World

A modern, responsive online bookstore website built using HTML, CSS, and JavaScript. The website provides an engaging shopping experience with a clean user interface, interactive features, and responsive design suitable for desktops, tablets, and mobile devices.

---

## 📖 Overview

Book World is an online bookstore that allows users to browse books, stationery products, and gift items through a modern and responsive interface.

The project was redesigned from a basic university assignment into a professional-looking e-commerce style website using only HTML, CSS, and JavaScript.
